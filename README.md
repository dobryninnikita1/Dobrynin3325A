# README

Описание проекта
ArtGuard — это децентрализованная платформа для художников, писателей, музыкантов и других творческих людей, позволяющая безопасно публиковать свои работы с подтверждением авторства и защитой авторских прав.
Проект создан для решения проблемы кражи и несанкционированного использования творческого контента в интернете. Каждая загруженная работа автоматически получает уникальный цифровой отпечаток (хеш), временную метку и неизменяемую запись в цепочке (на базе блокчейна или альтернативных решений), что служит юридически значимым доказательством авторства.
ArtGuard не является коммерческой платформой — это открытый инструмент, созданный для сообщества творцов, стремящихся сохранить контроль над своим творчеством.

Цели проекта
Защита авторских прав
Предоставить простой и доступный способ зафиксировать дату и факт создания работы.
Прозрачность
Все записи о публикациях неизменны и доступны для проверки.
Децентрализация
Избежать зависимости от централизованных платформ, которые могут удалить или ограничить доступ к работам.
Открытость
Исходный код проекта доступен всем — для аудита, доработки и распространения.
Простота использования
Минимальный интерфейс для загрузки, просмотра и проверки авторства — без сложных настроек.
